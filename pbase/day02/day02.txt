複合賦值運算符
  +=  -=  *=  /=  //=  %=  **=

  x += y 等同于 x = x + y
  x -= y 等同於 x = x - y

要求：執行此操作時變量應先存在

關係運算符：
    <  小於
    <=  小於等於
    >  大於
    >= 大於等於
    == 等於
    != 不等
說明：
  關係運算符返回布爾值類型的值（True/False)
  None 與任何對象都不相等，返回值為False
例：
  x = 68
  x > 60 #True
  80 > x > 60  #返回x是否在 (80~60)的區間
  60 < x < 80  #同上

生成對象的函數：
  float(obj) 用於字符串或數字轉換為浮點數，若不能出
  參數則返回0.0
  int(x[, base = 10])用數字或字符串轉為整數，如果不給出參數，則返回0
  bool(x) 用x生成一個布爾值(True/False)
  complex(r=0.0, i=0.0) 用數字生成一個複數(r = 實部, i = 虛部)
  str(obj='') | 用對象轉換為字符串

bool()函數返回假值False的情況：
False 邏輯假值
None  空值
0, 0.0, 0j 所有的數學零
""     空字符串
()     空字元
{}     空字典
[]     空列表
set    空集合

預置的數據類型函數：
   abs(x)  取x的絕對值
   round（number[,ndigis]）對數值進行四捨五入，
   ndigits是小數向右取整數的位數，負數表示向左取整
   pow(x y, z=None)  相當於x**y 或  x**y % z

示例：
  x = 123.456789
  round(x)   # 123
  round(x, 4) #123.4567
  round(x, -2) #100

基本輸出函數print
作用：
  將一系列的值以字符串形式輸出到標準輸出設備上
  ,默認為終端
格式：
  print(value, ..., sep='', end='\n', 
  file=sys.stdout, flash=False)

關鍵字參數的含意：
  sep: 兩個值之間的分隔符,默認為一個空格
  end: 輸出完畢後在劉末尾自動追加一個字符
  串,默認為換行符
  flush: 是否立即將流進行輸出
  file: 流對象,默認為sys.stdout

條件語句：
  if 語句
作用：
  讓程序根據條件選擇性的執行某些語句

語法：
    if真值表達式1:
       語句1
    elif 真值表達式2
       語句2
    elif 真值表達式3
       語句3
    ...
    elseL
       語句4
說明：
    elif子句可以有0個,1個或多個
    else子可最多只能有一個,也可以沒有
  
if語句嵌套：
   if語句本身是由多條語句組成的一條語句
   if語句可以作為語句嵌套到令一個語句的內部

if語句嵌套練習
   季節嵌套

   1-3春季
   4-6夏季
   ....
   要求：輸入月份打印季節(用if語句嵌套實現)

條件表達示：
語法：
  表達式1 if 真值表達式 else 表達式2
作用：
  如果真值表達式的布爾環境值為True, 表達式1執行並返回
對象,否則表達式2執行並返回對象

示例：
  見：if_express.py

取決對值：
 x = input("請輸入一個數：")
 x = int()
 x = x if x > 0 else -x # 等同於 x = abs(x)
